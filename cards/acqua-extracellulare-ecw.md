L'**Acqua Extra Cellulare** (ECW - Extracellular Water) è la quota di acqua corporea che si trova **al di fuori delle cellule**, negli spazi interstiziali, nel plasma e nella linfa.
Test 03

## Distribuzione dell'ECW

L'ECW rappresenta circa il **35-45% dell'acqua totale** (TBW) e comprende:

| Compartimento | % del peso corporeo |
|---------------|---------------------|
| Liquido interstiziale | 14% |
| Plasma sanguigno | 4% |
| Linfa | 1% |
| Liquido transcellulare | 1% |
| **Totale ECW** | ~20% |

## Valori di Riferimento

| Parametro | Valore Normale |
|-----------|----------------|
| ECW (% della TBW) | 38-45% |
| ECW (% peso corporeo) | 18-22% |
| Rapporto ECW/ICW | < 0.85 |
| Rapporto ECW/TBW | < 45% |

## Rapporto ECW/ICW

Il rapporto tra acqua extra e intracellulare è un indicatore chiave dello stato di idratazione:

| ECW/ICW | Interpretazione |
|---------|-----------------|
| 0.60-0.75 | Ottimale (normopeso) |
| 0.75-0.85 | Normale |
| > 0.85 | Ritenzione / Alterazione |
| < 0.60 | Possibile disidratazione |

## Cosa Indica un ECW Elevato

Un aumento dell'ECW può indicare:

| Condizione | Meccanismo |
|------------|------------|
| Ritenzione idrica | Accumulo di liquidi interstiziali |
| Edema | Trasudazione dai capillari |
| Infiammazione | Aumento permeabilità vascolare |
| Catabolismo | Perdita di massa cellulare |
| Malnutrizione | Riduzione BCM con espansione ECW |
| Insufficienza cardiaca | Congestione venosa |
| Insufficienza renale | Ridotta escrezione |
| Insufficienza epatica | Ridotta sintesi albumina |

## ECW e BIA

La BIA misura l'ECW attraverso la corrente a **bassa frequenza** (1-5 kHz):

```text
Bassa frequenza → Corrente passa solo nell'ECW
Alta frequenza → Corrente passa in ECW + ICW (TBW)
```

La differenza permette di calcolare l'ICW:
```text
ICW = TBW - ECW
```

## ECW nel Sovrappeso e Obesità

| Popolazione | ECW/ICW tipico |
|-------------|----------------|
| Normopeso | 0.70-0.75 |
| Sovrappeso | 0.80-0.90 |
| Obeso | > 0.85-1.0 |

Nei soggetti obesi, l'elevato ECW/ICW è associato a:
- Alterata regolazione metabolica
- Intolleranza al glucosio
- Riduzione della lipolisi
- Aumentato rischio di diabete

## ECW e Angolo di Fase

| PhA | ECW | Significato |
|-----|-----|-------------|
| Alto | Normale/Basso | Cellule sane, buona idratazione |
| Basso | Alto | Sofferenza cellulare, ritenzione |

L'angolo di fase basso con ECW alto indica catabolismo o infiammazione.

## Cause di ECW Aumentato

### Cause Fisiologiche
- Ciclo mestruale (fase luteale)
- Caldo ambientale
- Posizione eretta prolungata
- Fine gravidanza

### Cause Patologiche
- Scompenso cardiaco
- Cirrosi epatica
- Sindrome nefrosica
- Insufficienza venosa
- Linfedema
- Infiammazione cronica

## ECW e Hydragram

Nell'**Hydragram** dei software BIA (es. Akern):
- Vettore verso **l'alto** = ECW aumentata (iperidratazione)
- Vettore verso il **basso** = ECW ridotta (disidratazione)

## Monitoraggio dell'ECW

Indicazioni per il monitoraggio:

- Terapia diuretica
- Scompenso cardiaco
- Dialisi
- Cirrosi con ascite
- Diete ipocaloriche (per escludere perdita di BCM mascherata)

## In due parole

> "L'acqua extracellulare (ECW) è quella fuori dalle cellule: nel sangue, nella linfa, tra i tessuti. Dovrebbe essere circa il 40% dell'acqua totale. Se aumenta troppo, hai ritenzione idrica, edema o infiammazione. Il rapporto ECW/ICW sopra 0.85 è un campanello d'allarme: significa che c'è troppa acqua fuori rispetto a quella dentro le cellule."
